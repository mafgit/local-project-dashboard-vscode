**Effortlessly manage all your local projects in one modern, sleek dashboard — right inside VS Code.**
No more manually browsing folders or reopening recent projects.

[![ss](https://i.ibb.co/GvvsG0zS/ss.jpg)](https://ibb.co/xSSmwDxT)

## 🚀 Features

- ✅ **Unified Dashboard**: See all your local projects in one organized place.

- ✅ **Language Insights**: Detects languages, frameworks, and usage percentages automatically.

- ✅ **Quick Actions**:
    - Open in VS Code: Instantly open any project in a new VS Code window.
    - Open in Explorer: Click the folder icon to reveal the project in your system’s file explorer.
    - ✅ **Smart Scanning**: Add one or more base directories, and it will scan their immediate child folders for projects.
    - ✅ **Refresh Anytime**: Just click Scan/Refresh to update your project list.

## 🧭 Usage

1. Open the extension from sidebar inside VS Code (from the sidebar or command palette).
1. Add one or more base directories that contain your local projects.
1. Click Scan/Refresh to populate your project dashboard.
1. Open any project in a new VS Code window — or open its folder directly in Explorer.

That’s it. Your entire local workspace, always one click away. ⚡
